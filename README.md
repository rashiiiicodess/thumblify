# 🤖 Thumblify
An AI-powered thumbnail generation platform built using the **MERN** stack and **Google Gemini API** for fast, automated, and high-quality YouTube thumbnail creation. 
This project showcases full-stack development, RESTful APIs, asynchronous processing, and secure backend architecture.



## ✨ Features
- 🤖 AI-Powered Thumbnail Generation using Google Gemini API
- ⚡ Fast & Responsive Processing with optimized REST APIs
- 🔄 Asynchronous Backend Tasks for reduced latency
- 🔐 Secure API Key Management
- 📁 Cloud-Based Image Storage
- 📊 Scalable Backend Architecture


## 🛠️ Tech Stack
- **React.js**
- **Node.js**
- **Express.js**
- **MongoDB**
